# INSTALATION
## CLONE Repo
```
git clone https://github.com/idkraf/simple-crud.git
```
## INSTALL NPM
```
npm install express
```
## INSTALL NPM on client
- command on terminal vscode to folder client/src 
```
cd src/client
```
- install npm
```
npm install
```
- build npm
```
npm run build
```
- move back to main file with command cd .. and cd ..

# Run Script
```
npm start
```
or
```
node server.js
```
